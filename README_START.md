##运行步骤
1. 进到代码目录 
<pre><code>cd src/main/java</code></pre>
2. 执行
<pre><code>javac com/cyssxt/main/Application.java</code></pre>
3. 执行
<pre><code>java com.cyssxt.main.Application</code></pre>
4. 输入
<pre>
5 5 回车
1 2 N 回车
LMLMLMLMM 回车
3 3 E 回车
MMRMMRMRRM 回车
</pre>
输出：
<pre>
1 3 N
5 1 E
</pre>
5. 执行测试用例
<pre><code>
java com.cyssxt.main.Application
</code></pre>
